Curso de Bases de Datos

Descripción

En este curso se pretende que el estudiante conozca que son las Bases de datos, seguido se enfocará en lo que es una Base de Datos Relacional, por lo que se conocerá qué es un diagrama Entidad-Relacion, las Formas Normales que se aplican las BD RElacionales, Algebra Relacional y lenguaje SQL. En este curso aprenderás a dominar los temas difíciles de Base de Datos junto a tu profesora Fernanda Benavides.

Lo que aprenderás

El curso más Completo sobre Base de Datos ofrecido en EngineerLab

Conocerán desde lo básico hasta un nivel avanzado sobre Bases de Datos

Realizarán ejercicios prácticos y reales para obtener información de la Base de Datos

Conocerán desde lo básico hasta un nivel avanzado sobre Álgebra Relacional y lenguaje SQL

Descubrir cómo diseñar correctamente una Base de Datos Relacional
